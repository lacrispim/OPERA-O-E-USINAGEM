# **App Name**: FabriTrack

## Core Features:

- Production Registration: Register new production runs with fields for requesting factory, part name, material, and manufacturing time.
- Historical Records: View historical production records in a sortable, searchable table format, filterable by part name, date, and material.
- Production Parameter Recommendation: Using an LLM as a tool, this feature recommends optimized parameters based on historical data to improve manufacturing time.
- Dashboard Overview: Provide a dashboard with a summary view of recent production activities and key metrics.
- Data Export: Allow exporting production data in CSV format for external analysis.

## Style Guidelines:

- Background: Dark gray (#333333) to provide a sophisticated, low-distraction environment.
- Primary: Pastel blue (#A0D2EB) for buttons, icons, and highlights, offering a calm yet distinct call to action.
- Accent: Light gray (#EEEEEE) for text and data displays, ensuring readability against the dark background.
- Font: 'Inter', a sans-serif font, for a clean, modern, and readable UI. Suitable for both headings and body text.
- Retractable Sidebar: A sidebar that expands on hover, containing navigation options like Dashboard, Historical Records, and Optimize Production.
- Use simple, line-based icons in pastel blue to represent different actions and categories, maintaining a clean and consistent aesthetic.
